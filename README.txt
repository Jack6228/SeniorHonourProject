# Column Descriptions:

U_*: Parameters as measured in the u-band
R_*: Parameters as measured in the r-band
RA: Right ascension of barycenter (J2000)
Dec: Declination of barycenter (J2000)
*_MAG_COG: Curve of growth magnitude
*_MAG_APAUTO: Aperture magnitude corresponding to MAG_AUTO for star
*_MAGERR_*: RMS error for MAG_*
*_IQ: FWHM of a point source at this position ["]
*_A_WORLD: Profile RMS along major axis (world units)
*_B_WORLD: Profile RMS along minor axis (world units)
*_THETA_J2000: Position angle (east of north) (J2000)
*_ISOAREA_IMAGE: Isophotal area above analysis threshold
*_MU_MAX: Peak surface brightness above background
*_FLUX_RADIUS: Half-light radius ["]
*_FLAGS: Extraction flags
*_MAG_2ARC: Aperture magnitude in a 2 arsecond aperture
*_MAGLIM_*: 5-sigma point source limiting magnitude for MAG_*
*_MAG_APER_5: Fixed aperture magnitude 5 pixels diameter
*_MAG_APER_10: Fixed aperture magnitude 10 pixels diameter
*_MAG_APER_15: Fixed aperture magnitude 20 pixels diameter
*_MAG_APER_20: Fixed aperture magnitude 40 pixels diameter
